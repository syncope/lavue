.. _configuration:

Configuration
=============


The *Configuration* button appears in the top-right corner in the expert mode, i.e.

.. code-block:: console

  $ lavue -m expert

It opens the configuration dialog where the user select matching LaVue **settings**.

*    On normal exit of LaVue the settings are **stored** at `$HOME/.config/DESY/LaVue.conf`
*    The settings are **loaded** back from: `$HOME/.config/DESY/LaVue.conf`  when the user starts LaVue.
